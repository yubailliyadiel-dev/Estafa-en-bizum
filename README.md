La estafa, que consiste en enviar dinero desde una tarjeta robada a un usuario aleatorio

# ¿En qué consiste la estafa?

La estafa de Bizum suele consistir en que el estafador envía o pide una solicitud de dinero haciéndola pasar por un pago, engañando a la víctima para que la acepte y le transfiera dinero.
También usan mensajes falsos de bancos o compradores para obtener el código Bizum o convencerte de “devolver” un pago inexistente.

# La estafa de Bizum paso a paso:

1- El estafador se hace pasar por comprador/vendedor o por tu banco y gana tu confianza.

2- Te envía una solicitud de dinero diciendo que es un pago o que debes aceptarla para cobrar.

3- Al aceptar la solicitud, eres tú quien envía el dinero al estafador.

4- El estafador desaparece y el dinero no se puede recuperar fácilmente.

# ¿Qué debo hacer?

1- Contacta inmediatamente con tu banco y explica que ha sido una estafa por Bizum (a veces pueden bloquear al receptor).

2- Denuncia en la policía y guarda pruebas: chats, número, justificante Bizum, capturas.

3- Bloquea al estafador y no vuelvas a responderle.

4- Cambia contraseñas y revisa que no tengan más datos tuyos.
